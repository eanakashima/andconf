Welcome to &:conf

&:conf, pronounced "and conf," is a code retreat and an unconference for intersectional feminist web developers, August 14-17, 2015 in the woods of Western Sonoma County. Saturday’s code retreat will be a day of pair programming on one kata where each hour we throw our code away and work with a new person. Sunday’s unconference will have two tracks: yak and hack, where we'll talk about technologies, programming, careers, and culture.
